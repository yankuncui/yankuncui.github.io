一、枚举算法的思想

枚举算法的定义

在进行归纳推理时，如果逐个考察了某类事件的所有可能情况，因而得出一般结论，那么该结论是可靠 的，
这种归纳方法叫做枚举法。

枚举算法的思想

将问题的所有可能的答案一一列举，然后根据条件判断此答案是否合适，保留合适的，舍弃不合适的。

使用枚举算法解题的基本思路

（1）确定枚举对象、范围和判定条件。
（2）逐一枚举可能的解并验证每个解是否是问题的解。

枚举算法步骤

（1）确定解题的可能范围，不能遗漏任何一个真正解，同时避免重复。
（2）判定是否是真正解的方法。
（3）为了提高解决问题的效率，使可能解的范围将至最小。

二、算法实例

题目描述
观察数字：12321，123321都有一个共同的特征，无论从左往右读，还是从右往左读，都是相同的，
上述数字称为回文数字。

现在要从5位或6位的十进制数字中找出各个位数之后等n的回文数字。

输入格式
输入一个整数n(10<n<=100)

输出格式
输出所有个位数之和等于n的5位和6位整数，每个数字占一行，数字从小到大的顺序排列。
如果没有满足的数字，则输出-1。
样例输入
48

样例输出
699996
789987
798897
879978
888888
897798
969969
978879
987789
996699

代码
#include<bits/stdc++.h>
using namespace std;
int n;
int digit[6];
bool jug(int x){
int m=0,sum=0;
while(x){
digit[m++]=x%10;
sum+=x%10;
x/=10;
}
if(sum!=n){
return false;
}
for(int i=0;i<m/2;i++){
if(digit[i]!=digit[m-1-i])
return false;
}
return true;
}
int main(){
bool f=false;
cin>>n;
for(int i=10000;i<1000000;i++){
if(jug(i)){
cout<<i<<endl;
f=true;
}
}
if(!f){
cout<<-1<<endl;
}
return 0;
}

题目描述
方程a^2+b^2+c^2=n，其中0<a<b<c。请你求出它的所有解，结果按照
a的值从小到大输出。

输入格式
输入一个整数n，为方程等号右边的整数。

输出格式
输出若干行，每行包括三个空格隔开的整数，分别为a,b,c的值。

样例输入
1000

样例输出
6 8 30
10 18 24

代码
#include<bits/stdc++.h>
using namespace std;
int main(){
int n;
cin>>n;
for(int a=1;a * a<=n;a++){
for(int b=a+1;a * a+b * b<=n;b++){
for(int c=b+1;a * a+b * b+c * c<=n;c++){
if(a * a+b * b+c * c==n)
cout<<a<<” “<<b<<” “<<c<<endl;
}
}
}
return 0;
}

题目描述
给定一个n x m的矩阵A。求A中的一个非空子矩阵，使这个子矩阵
中的元素和最大。其中，A的子矩阵是指A中行和列均连续的一部分。

输入格式
输入的第一行包含两个整数
n,m(n>=1,m<=50),分别表示矩阵A的行数和列数，接下来n行，每行
m个整数，表示矩阵Aij(-1000<=Aij<=1000)。

输出格式
输出一行，每包含一个整数，表示A中最大的子矩阵中的元素和

样例输入
3 3
2 -4 1
-1 2 1
4 -2 2

样例输出
6

代码
#include<bits/stdc++.h>
using namespace std;
int A[55][55];
int main(){
int n,m,ans;
cin>>n>>m;
ans=-1005;
for(int i=0;i<n;i++){
for(int j=0;j<m;j++){
cin>>A[i][j];
}
}
for(int i=0;i<n;i++){
for(int j=i;j<n;j++){
for(int k=0;k<m;k++){
for(int l=k;l<m;l++){
int tmp=0;
for(int p=i;p<=j;p++){
for(int q=k;q<=l;q++){
tmp+=A[p][q];
}
}
if(tmp>ans){
ans=tmp;
}
}
}
}
}
cout<<ans<<endl;
return 0;
}
